# OpenClaw Tools v1.0.0

Initial release with 8 essential tools for monitoring, securing, and optimizing your OpenClaw deployment.

## What's Included

### Security
- oct-security-audit - Full security audit with pattern detection
- oct-security-fast - Quick 30-second security health check

### Monitoring
- oct-daily-audit - Daily system optimization report
- oct-monitor - Real-time system monitor
- oct-cost-track - API cost tracking by model/provider

### Tracking
- oct-token-track - Daily token usage with model breakdown
- oct-token-summary - Aggregate token reports across date ranges
- oct-health-check - Complete system health diagnostic
